# Auto Motors Website

Статический 5-страничный сайт-визитка для автосервиса на русском языке.

## Страницы

- `index.html` — главная
- `services.html` — услуги
- `about.html` — о компании
- `gallery.html` — галерея работ
- `contacts.html` — контакты и форма записи

## Настройка

Замените телефон, WhatsApp, адрес, карту и фотографии на реальные данные сервиса. Форма сейчас работает как фронтенд-демо; для реальных заявок подключите backend, CRM, Telegram-бота или почтовый сервис.
